# 布局规范

## 应用壳（App Shell）

```
┌─────────────────────────────────────────────────────────────┐
│  Titlebar (Tauri custom, 40px)                              │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│ Sidebar  │  Main Canvas                                     │
│ 260px    │  flex-1                                          │
│          │                                                  │
│ Workspace│  首页: 卡片网格                                   │
│ 列表     │  详情: 控制面板                                   │
│          │                                                  │
│ + 添加   │                                                  │
│          │                                                  │
├──────────┴──────────────────────────────────────────────────┤
│  Status Bar (可选, 24px): 版本号 / 主题切换                    │
└─────────────────────────────────────────────────────────────┘
```

- Sidebar 固定 260px，可折叠至 64px（仅图标）
- Main Canvas 最小宽度 640px
- 窗口默认 1280×800，最小 960×640

## 首页：Workspace 卡片网格

```
┌─────────────────────────────────────────┐
│  工作区                          [+ 添加] │
│  管理你的 MCP 运行时                      │
├─────────────────────────────────────────┤
│                                         │
│  ┌──────────────┐  ┌──────────────┐     │
│  │ ● my-project │  │ ○ api-server │     │
│  │ ~/code/my... │  │ ~/code/api.. │     │
│  │ FRP · 运行中  │  │ 未配置        │     │
│  │ https://...  │  │              │     │
│  └──────────────┘  └──────────────┘     │
│                                         │
└─────────────────────────────────────────┘
```

卡片内容（从上到下）：
1. 状态灯 + 名称（semibold）
2. 路径（mono, muted, 单行截断）
3. 隧道类型 + 运行状态（badge）
4. Endpoint 摘要（mono, 单行截断，hover 显示完整）

卡片网格：`grid-cols-1 md:grid-cols-2 xl:grid-cols-3`，gap 16px

## 间距系统

基于 4px 阶梯：`4, 8, 12, 16, 24, 32, 48, 64`

| 场景 | 间距 |
|------|------|
| 卡片内 padding | 20px (16+4) |
| 卡片间距 | 16px |
| 区块间距 | 24px |
| 页面边距 | 24px (sidebar 内 16px) |
| 表单字段间距 | 12px |

## 圆角

| 元素 | 圆角 |
|------|------|
| 卡片 | 14px (lg) |
| 按钮 | 10px (md) |
| 输入框 | 10px (md) |
| Badge | 6px (sm) |
| 状态灯 | full |

## Z-index

dropdown: 100 → sticky: 200 → modal: 300 → toast: 400 → tooltip: 500
